Plotting pedigrees with plotPedigree()

library(ggBGmisc)
library(BGmisc)

Here we will show how to visualize pedigrees using the ggPedigree() function. This function is built in ggplot2. We will use the potter and hazard datasets from the BGmisc package as examples.

data("potter")
ggPedigree(potter, famID_col = "famID", personID_col = "personID", code_male = 1)

data("hazard")
ggPedigree(hazard, famID_col = "famID", personID_col = "ID", code_male = 0)
